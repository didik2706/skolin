## Skolin
Aplikasi big data sekolah untuk mempermudah mencari sekolah favorite.

#### Installasi

1.  clone dulu dari github
2.  npm install
3.  composer install
4.  import database <tigalic_skolin.sql>
5.  jangan Lupa untuk mencopy .env
